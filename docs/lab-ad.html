Lab Active Directory — Détection & Réponse

Ce laboratoire simule un système d’information d’entreprise afin d’étudier la détection et la réponse face à des attaques ciblant Active Directory.

Architecture

  • Contrôleur de domaine Windows Server
  • Poste client utilisateur
  • Firewall de segmentation
  • Machine attaquante dédiée

Objectifs pédagogiques

  • Construire une infrastructure réaliste
  • Rendre le système observable
  • Détecter une attaque
  • Mettre en place une réponse sécurité
