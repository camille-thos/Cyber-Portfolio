# Active Directory – Attaque & Défense

## Objectif

Comprendre les mécanismes d’attaque courants sur un environnement Active Directory et analyser les traces laissées dans les journaux système.

## Architecture du Lab

- Windows Server 2022 – Contrôleur de domaine
- Windows 10 – Poste client
- Kali Linux – Machine d’attaque

## Étape 1 – Mise en place

Installation du rôle AD DS
Création d’utilisateurs
Configuration des politiques d’audit

## Étape 2 – Attaque simulée
